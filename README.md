# gestor-pedidos

Projeto pessoal desenvolvido com Django para simular um sistema simples de gestão de pedidos.

Personal project developed using Django to simulate a simple order management system.

## Tecnologias / Tech Stack
- Python / Django
- MySQL
- Pandas

## Funcionalidades Planejadas / Planned Features
- Autenticação e controle de acesso por perfis usando o sistema de permissões e grupos do Django
- Authentication and role-based access control using Django permissions and groups
